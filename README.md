# devcraftpablo.online — Portfolio Personal

Portfolio personal de Pablo Proboste, desarrollador frontend freelance.

## Stack
- React + Vite
- Tailwind CSS
- Lucide React (íconos)
- Vercel (deploy)

## Instalación
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Deploy automático en Vercel al pushear a `main`.
